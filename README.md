# 📈 Live Stock Price Dashboard

A real-time Stock Price Dashboard built with Python, Streamlit, and Plotly.

## 🔧 Tech Stack
- Python
- Streamlit
- Plotly
- yfinance (Yahoo Finance)
- Streamlit-Autorefresh

## 📊 Features
- Live real-time stock prices
- Candlestick chart (5-min intervals)
- Day High & Day Low metrics
- Volume bar chart
- Price % Change indicator
- Raw data table
- Auto-refreshes every 60 seconds

## 💹 Supported Stocks
- Apple (AAPL)
- Google (GOOGL)
- Microsoft (MSFT)
- Tesla (TSLA)

## 🚀 How to Run Locally
1. Clone this repository
2. Install dependencies:
   pip install -r requirements.txt
3. Run the dashboard:
   streamlit run finance_dashboard.py
4. Open browser at http://localhost:8501

## 📡 Data Source
Data is fetched live from Yahoo Finance via the yfinance library.
No API key required.
